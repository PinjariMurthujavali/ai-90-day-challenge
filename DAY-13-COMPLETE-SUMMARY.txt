## 📞 TROUBLESHOOTING

**Problem:** Redis connection failed
```
Solution: Make sure redis-server is running in another terminal
redis-server
```

**Problem:** ModuleNotFoundError: graphene
```
Solution: Install dependencies
pip install -r requirements.txt
```

**Problem:** GraphQL endpoint returning errors
```
Solution: Check if Redis is running + all Python files present
```

**Problem:** Queries still slow (no caching)
```
Solution: Warm cache
curl -X POST http://localhost:5000/api/v1/cache/warm
```
